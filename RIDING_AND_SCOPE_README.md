# 快乐恶魂骑乘和瞄准镜系统使用指南

## 🎮 新功能概览

现在玩家可以：
1. 🐴 **骑乘快乐恶魂** - 像骑马一样骑在快乐恶魂上
2. 🎯 **使用瞄准镜** - 手持瞄准镜长按右键进行瞄准
3. 🔥 **发射火球** - 松开右键后让快乐恶魂发射火球到瞄准位置

## 📋 快乐恶魂瞄准镜

### 物品介绍
- **名称**：快乐恶魂瞄准镜 (Happy Ghast Scope)
- **功能**：骑乘快乐恶魂时，可以瞄准并发射火球
- **特效**：使用时具有望远镜效果（视野放大）

### 合成配方

```
[ 玻璃 ] [ 玻璃 ] [ 玻璃 ]
[ 玻璃 ] [末影眼] [ 玻璃 ]
[      ] [ 木棍 ] [      ]
```

**材料**：
- 5个玻璃 (Glass)
- 1个末影之眼 (Ender Eye)
- 1个木棍 (Stick)

**产出**：1个快乐恶魂瞄准镜

## 🐴 骑乘快乐恶魂

### 如何骑乘

1. **准备工作**
   - 确保你的手是空的（不持有任何物品）
   - 找到一只快乐恶魂

2. **上马**
   - 对着快乐恶魂右键点击
   - 不要按住Shift键
   - 成功后会显示提示：`"你骑上了快乐恶魂！按Shift下马"`

3. **下马**
   - 按住Shift键即可下马
   - 或者从高处掉落时会自动下马

### 骑乘注意事项

- ✅ 快乐恶魂可以飞行，骑乘时你也会随之飞行
- ✅ 一次只能有一个玩家骑乘
- ⚠️ 按住Shift键会打开快乐恶魂的GUI而不是骑乘
- ⚠️ 手持物品时无法骑乘

## 🎯 使用瞄准镜发射火球

### 操作步骤

**第一步：骑乘**
1. 空手右键点击快乐恶魂
2. 成功骑上后会收到提示消息

**第二步：装备瞄准镜**
1. 从背包或快捷栏中拿出快乐恶魂瞄准镜
2. 将其装备到手上

**第三步：瞄准**
1. 按住右键开始瞄准
2. 视野会像使用望远镜一样放大
3. 移动鼠标调整瞄准方向
4. 至少按住0.5秒（10 ticks）才能发射

**第四步：发射**
1. 松开右键
2. 快乐恶魂会立即向你瞄准的方向发射火球
3. 收到提示消息：`"火球发射！"`

### 发射参数

火球的威力**根据快乐恶魂的等级决定**：

| 快乐恶魂等级 | 火球威力 | 爆炸效果 |
|-------------|---------|---------|
| 1级 | 1 | 小型爆炸 |
| 2级 | 2 | 较小爆炸 |
| 3级 | 3 | 中等爆炸 |
| 4级 | 4 | 较大爆炸 |
| 5级 | 5 | 大型爆炸 |
| 6级 | 6 | 超大爆炸 |

**重要提示**：
- 🔥 火球威力可在配置文件中自定义
- ⚡ 没有冷却时间限制（玩家控制）
- 🎯 精准度取决于你的瞄准技术
- 💨 火球会沿直线飞行

## 🎮 游戏玩法技巧

### 基础战术

**1. 高空轰炸**
- 骑乘快乐恶魂飞到高空
- 俯瞰地面寻找目标

**3. 团队配合**

- 其他玩家在地面引导目标

## 🎓 常见问题

### Q1：为什么无法骑乘快乐恶魂？
**A1：** 检查以下几点：
- ✔️ 手是否是空的（不持有任何物品）
- ✔️ 是否没有按住Shift键
- ✔️ 快乐恶魂是否已经有其他乘客

### Q2：为什么瞄准镜无法使用？
**A2：** 检查以下几点：
- ✔️ 你必须正在骑乘快乐恶魂
- ✔️ 手中必须拿着瞄准镜
- ✔️ 按住右键至少0.5秒

### Q3：火球威力太小/太大怎么办？
**A3：** 修改配置文件：
- 打开 `.minecraft/config/chest-on-ghast.json`
- 找到对应等级的 `fireballPower` 参数
- 调整数值后重启游戏

### Q4：可以在快乐恶魂上使用其他物品吗？
**A4：** 可以！
- ✅ 瞄准镜是专用的发射火球工具
- ✅ 你可以手持其他物品进行其他操作
- ✅ 例如：喝药水、使用弓箭等

### Q5：火球会伤害快乐恶魂自己吗？
**A5：** 
- 🛡️ 通常不会，快乐恶魂是火球的发射者
- ⚠️ 但爆炸范围伤害可能影响周围实体
- 💡 建议保持安全距离

## 🔧 配置参考

快乐恶魂的火球威力在配置文件中可调整：

```json
{
  "levels": {
    "1": {
      "fireballPower": 1,
      "attackCooldown": 60
    },
    "6": {
      "fireballPower": 6,
      "attackCooldown": 15
    }
  }
}
```

**注意**：玩家控制发射时不受 `attackCooldown` 限制！
