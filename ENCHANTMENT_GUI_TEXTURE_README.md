## 临时方案

如果暂时没有纹理文件，代码会尝试使用原版纹理。你可以：

1. **使用原版纹理（推荐）**：代码已经设置为使用 `minecraft:textures/gui/container/generic_54.png`
   
2. **创建自定义纹理**：
   - 可以复制Minecraft原版的 `generic_54.png` 并修改
   - 或者使用任何图像编辑软件创建 256x256 的PNG
   - 只需要绘制背景框和槽位即可

3. **素材参考**：
   - 参考原版的箱子GUI、熔炉GUI等
   - 背景通常是浅灰色
   - 槽位是深灰色的18x18方格

## 颜色参考
- 背景：#C6C6C6（浅灰）
- 边框：#373737（深灰）
- 槽位：#8B8B8B（中灰）
- 槽位边框：#FFFFFF（白色）

## 注意事项
- 纹理文件必须是PNG格式
- 尺寸必须是256x256像素（即使只使用176x166区域）
- 透明通道可选
- 如果缺少纹理文件，GUI会显示空白但仍可正常使用
