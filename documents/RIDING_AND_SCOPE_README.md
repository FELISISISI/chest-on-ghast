# 快乐恶魂瞄准镜系统使用指南

## 🎮 新功能概览

现在玩家可以：
1. 🐴 **骑乘快乐恶魂** - 给快乐恶魂装备挽具后即可骑乘（原版功能）
2. 🎯 **使用瞄准镜** - 骑乘时手持瞄准镜长按右键进行瞄准
3. 🔥 **发射火球** - 松开右键后让快乐恶魂发射火球到瞄准位置

## 📋 快乐恶魂瞄准镜

### 物品介绍
- **名称**：快乐恶魂瞄准镜 (Happy Ghast Scope)
- **功能**：骑乘快乐恶魂时，可以瞄准并发射火球
- **特效**：使用时具有望远镜效果（视野放大）

### 合成配方

```
[ 玻璃 ] [ 玻璃 ] [ 玻璃 ]
[ 玻璃 ] [末影眼] [ 玻璃 ]
[      ] [ 木棍 ] [      ]
```

**材料**：
- 5个玻璃 (Glass)
- 1个末影之眼 (Ender Eye)
- 1个木棍 (Stick)

**产出**：1个快乐恶魂瞄准镜

## 🐴 骑乘快乐恶魂

### 准备工作

**重要提示**：快乐恶魂使用原版的骑乘机制，需要先装备挽具！

### 如何骑乘

**第一步：装备挽具**
1. 准备一个挽具 (Saddle)
2. 对着快乐恶魂右键使用挽具
3. 挽具会装备到快乐恶魂身上

**第二步：骑乘**
1. 对着已装备挽具的快乐恶魂右键点击
2. 你会自动骑上快乐恶魂
3. 现在可以控制它的移动方向

**第三步：下马**
- 按住Shift键即可下马
- 或者从高处掉落时会自动下马

### 骑乘注意事项

- ⚠️ **必须先装备挽具才能骑乘**
- ✅ 快乐恶魂可以飞行，骑乘时你也会随之飞行
- ✅ 一次只能有一个玩家骑乘
- ✅ 使用W/A/S/D键控制方向
- ✅ 空格键上升，Shift键下马

## 🎯 使用瞄准镜发射火球

### 操作步骤

**第一步：装备挽具**
1. 对快乐恶魂右键使用挽具
2. 确保挽具已装备

**第二步：骑乘**
1. 右键点击快乐恶魂
2. 成功骑上后可以控制移动

**第三步：装备瞄准镜**
1. 从背包或快捷栏中拿出快乐恶魂瞄准镜
2. 将其装备到手上

**第四步：瞄准**
1. 按住右键开始瞄准
2. 视野会像使用望远镜一样放大
3. 移动鼠标调整瞄准方向
4. 至少按住0.5秒（10 ticks）才能发射

**第五步：发射**
1. 松开右键
2. 快乐恶魂会立即向你瞄准的方向发射火球
3. 收到提示消息：`"火球发射！"`

### 发射参数

火球的威力**根据快乐恶魂的等级决定**：

| 快乐恶魂等级 | 火球威力 | 爆炸效果 |
|-------------|---------|---------|
| 1级 | 1 | 小型爆炸 |
| 2级 | 2 | 较小爆炸 |
| 3级 | 3 | 中等爆炸 |
| 4级 | 4 | 较大爆炸 |
| 5级 | 5 | 大型爆炸 |
| 6级 | 6 | 超大爆炸 |

**重要提示**：
- 🔥 火球威力可在配置文件中自定义
- ⚡ 没有冷却时间限制（玩家控制）
- 🎯 精准度取决于你的瞄准技术
- 💨 火球会沿直线飞行

## 🎮 游戏玩法技巧

### 基础战术

**1. 高空轰炸**
- 骑乘快乐恶魂飞到高空
- 俯瞰地面寻找目标
- 使用瞄准镜精确打击

**2. 快速机动**
- 利用快乐恶魂的飞行能力快速移动
- 在空中调整位置寻找最佳射击角度
- 打了就跑，保持安全距离

**3. 团队配合**
- 一个玩家骑乘控制火球
- 其他玩家在地面引导目标
- 形成空地协同作战

### 高级技巧

**1. 预判射击**
- 针对移动目标，提前瞄准移动方向
- 考虑火球飞行时间
- 练习提前量

**2. 狙击模式**
- 在远距离使用瞄准镜
- 利用视野放大效果精确瞄准
- 适合对付固定目标

**3. 连续射击**
- 快速瞄准->发射->重新瞄准
- 形成火力压制
- 适合对付群体敌人

## ⚔️ 战斗应用

### PvE（玩家对怪物）

**对付单体敌人**

**对付群体敌人**
- 利用爆炸范围伤害多个目标
- 瞄准怪物集中的位置
- 配合地形造成额外伤害

### PvP（玩家对玩家）

**攻击策略**
- 突袭：快速接近，近距离轰炸
- 游击：远距离骚扰，打了就跑
- 围困：高空封锁，限制敌人移动

**防守策略**
- 空中巡逻：监视领地，及时反击
- 要塞守卫：高空射击接近的敌人
- 逃生：快速撤离危险区域

## 🎓 常见问题

### Q1：为什么无法骑乘快乐恶魂？
**A1：** 检查以下几点：
- ✔️ 快乐恶魂是否已装备挽具
- ✔️ 快乐恶魂是否已经有其他乘客
- ✔️ 是否在正确的位置点击（实体中心）

### Q2：如何给快乐恶魂装备挽具？
**A2：** 
- 手持挽具
- 对着快乐恶魂右键点击
- 挽具会自动装备上去

### Q3：为什么瞄准镜无法使用？
**A3：** 检查以下几点：
- ✔️ 你必须正在骑乘快乐恶魂
- ✔️ 手中必须拿着瞄准镜
- ✔️ 按住右键至少0.5秒

### Q4：火球威力太小/太大怎么办？
**A4：** 修改配置文件：
- 打开 `.minecraft/config/chest-on-ghast.json`
- 找到对应等级的 `fireballPower` 参数
- 调整数值后重启游戏

### Q5：可以在快乐恶魂上使用其他物品吗？
**A5：** 可以！
- ✅ 瞄准镜是专用的发射火球工具
- ✅ 你可以手持其他物品进行其他操作
- ✅ 例如：喝药水、使用弓箭等

### Q6：火球会伤害快乐恶魂自己吗？
**A6：** 
- 🛡️ 通常不会，快乐恶魂是火球的发射者
- ⚠️ 但爆炸范围伤害可能影响周围实体
- 💡 建议保持安全距离

### Q7：挽具可以取回吗？
**A7：**
- ✅ 可以！使用原版机制取回挽具
- 🔧 对着已装备挽具的快乐恶魂右键
- 📦 或者快乐恶魂死亡时会掉落挽具

## 🔧 配置参考

快乐恶魂的火球威力在配置文件中可调整：

```json
{
  "levels": {
    "1": {
      "fireballPower": 1,
      "attackCooldown": 60
    },
    "6": {
      "fireballPower": 6,
      "attackCooldown": 15
    }
  }
}
```

**注意**：玩家控制发射时不受 `attackCooldown` 限制！
